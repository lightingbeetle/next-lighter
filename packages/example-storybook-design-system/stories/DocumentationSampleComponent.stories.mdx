## Usage

Use this component to render a paragraph with custom text by passing it as `children` prop to the component.
Optionally specify `variant` prop to control the size of font in the paragraph.

## Variants

### Small

<DocumentationSampleComponent variant="small">
  This is a paragraph in small variant.
</DocumentationSampleComponent>

### Large

<DocumentationSampleComponent variant="small">
    This is a paragraph in large variant.
</DocumentationSampleComponent>
