# Bar

Bar is component to create horizontal flexible width layouts.

## Example

<Bar>
  <BarItem>
    <Rectangle>Item</Rectangle>
  </BarItem>
  <BarItem isFilling>
    <Rectangle>Filling Item</Rectangle>
  </BarItem>
  <BarItem>
    <Rectangle>Item</Rectangle>
  </BarItem>
</Bar>
