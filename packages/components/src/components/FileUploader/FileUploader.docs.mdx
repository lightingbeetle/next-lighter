### Vlastné nastavenia pre React-dropzone

V tejto ukážke `FileUploader` prijíma iba súbory vo formáte `.pdf` s maximálnou veľkosťou `0kb`. Po vybratí akéhokoľvek `.pdf` súboru sa tak zobrazí preddefinované chybové hlásenie o tom, že vybraný súbor je väčší ako nastavený limit.

<Preview>
  <FileUploader
    id="file-uploader-example-7"
    accept={{
      "application/pdf": [".pdf"],
    }}
    dropzoneOptions={{ maxSize: 0 }}
  >
    <FileUploaderPreview />
    <FileUploaderArea />
    <FileUploaderError />
  </FileUploader>
</Preview>

Toto preddefinované chybové hlásenie je možné zmeniť pomocou `fileSizeErrorMessage` propu

<Preview>
  <FileUploader
    id="file-uploader-example-8"
    accept={{
      "application/pdf": [".pdf"],
    }}
    dropzoneOptions={{ maxSize: 0 }}
  >
    <FileUploaderPreview />
    <FileUploaderArea />
    <FileUploaderError
      fileSizeErrorMessage={
        "Custom error message that is shown when uploaded file has bigger size than is accepted."
      }
    />
  </FileUploader>
</Preview>

V tejto ukážke `FileUploader` prijíma iba súbory vo formáte `.jpeg`. Pre vyvolanie preddefinovaného chybového hlásenia o nesprávnom formáte súboru je potrebné súbor pretiahnuť do označenej oblasti, pretože pri výbere z prieskumníka, ktorý sa zobrazí po kliknutí na `FileUploader` sa zobrazujú iba súbory s povoleným formátom.

<Preview>
  <FileUploader
    id="file-uploader-example-9"
    accept={{ "image/jpg": [".jpg", ".jpeg"] }}
  >
    <FileUploaderPreview />
    <FileUploaderArea />
    <FileUploaderError />
  </FileUploader>
</Preview>

Toto preddefinované chybové hlásenie je možné zmeniť pomocou `acceptedFormatsErrorMessage` propu

<Preview>
  <FileUploader
    id="file-uploader-example-10"
    accept={{ "image/jpg": [".jpg", ".jpeg"] }}
  >
    <FileUploaderPreview />
    <FileUploaderArea />
    <FileUploaderError
      acceptedFormatsErrorMessage={
        "Custom error message that is shown when uploaded file has different format than is accepted."
      }
    />
  </FileUploader>
</Preview>
