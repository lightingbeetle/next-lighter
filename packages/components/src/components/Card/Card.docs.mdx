Navigačný komponent. Môže obsahovať obrázok, nadpis, text aj button/link.

Karta slúži ako vstupný bod k podrobnejším informáciám. Karty môžu obsahovať bohatý obsah, ako sú fotografie, text, odkaz, zoznam, tabuľky,..

### Ukážka

<Preview>
  <Card>
    <CardSection>
      <CardTitle>Simple card</CardTitle>
      <p>Card Content</p>
    </CardSection>
  </Card>
</Preview>

### Použitie

<p>
  Pri klikateľnej kartičke je potrebné manuálne poslať do <code>Card</code>
  komponentu <code>isClickable</code> prop, vďaka ktorému kartička nadobudne štýly,
  ktoré signalizujú, že je klikateľná.
</p>
<p>
  Pre nadpis kartičky používajte <code>CardTitle</code> komponent. Prebieha tu
  kontrola, či je použitý v kombinácii s <code>CardAction</code> komponentom, na
  základe ktorej sa priradzuje <code>className</code>, ktorá je potrebná pre
  správne fungovanie klikateľnej kartičky.
</p>
<p>
  Pri klikateľnej kartičke je nutné použiť <code>CardAction</code>
  komponent. Slúži ako wrapper nad <code>Linkom</code> alebo <code>
    Buttonom
  </code>, ktorý je do tohto komponentu poslaný. Následne je pomocou štýlov upravený
  tak, aby bola celá kartička klikateľná.
</p>

### JS module reference

All card sections elements with `[data-card-section]` attribute will be initialised automatically.

#### Custom initialisation

```js
const instance = new window.myApp.CardSection(
  document.getElementById("my-card-section"),
  config
);
```
