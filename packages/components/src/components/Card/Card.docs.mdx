# Card

Navigačný komponent. Môže obsahovať obrázok, nadpis, text aj button/link.

Karta slúži ako vstupný bod k podrobnejším informáciám. Karty môžu obsahovať bohatý obsah, ako sú fotografie, text, odkaz, zoznam, tabuľky,..

### Použitie

Pri klikateľnej kartičke je potrebné manuálne poslať do `Card`
komponentu `isClickable` prop, vďaka ktorému kartička nadobudne štýly,
ktoré signalizujú, že je klikateľná.

Pre nadpis kartičky používajte `CardTitle` komponent. Prebieha tu
kontrola, či je použitý v kombinácii s `CardAction` komponentom, na
základe ktorej sa priradzuje `className`, ktorá je potrebná pre
správne fungovanie klikateľnej kartičky.

Pri klikateľnej kartičke je nutné použiť `CardAction` komponent. Slúži ako wrapper nad `Link` alebo `Button`, ktorý je do tohto komponentu poslaný. Následne je pomocou štýlov upravený tak, aby bola celá kartička klikateľná.

### JS module reference

All card sections elements with `[data-card-section]` attribute will be initialised automatically.

#### Custom initialisation

```js
const instance = new window.myApp.CardSection(
  document.getElementById("my-card-section"),
  config
);
```
