## Customize path to sprite

In React enviroment, path to sprite can be customized globally via [Context API](https://reactjs.org/docs/context.html) and `<IconProvider />`.

```jsx
// This is same as defining every icon as <Icon spritePath="/my-sprite.svg" />
function App() {
  return (
    <IconProvider spritePath="/my-sprite.svg">
      <Icon name="home" />
    </IconProvider>
  );
}
```

## Accessibility

The Icon component is implemented by the [W3C](https://www.w3.org/wiki/SVG_Accessibility) and [Accessible SVG Icons by Chris Coyier](https://css-tricks.com/accessible-svg-icons/).

Current implementation does not support:

- Description (represented in sprite as xml `<desc />` markup).
